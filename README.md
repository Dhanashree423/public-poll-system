# PulseVote (Anonymous & Social)

## Overview
Public Poll System is a web-based platform that allows users to create and participate in anonymous polls.  
The system is designed to encourage open and unbiased opinions without requiring user identification.

---

## Key Features
- Create public polls
- Anonymous voting
- Real-time vote count 
- Responsive UI
- Clean, minimal interface
- Social discussion-based concept

---

## Problem Statement
Many users hesitate to express opinions due to identity exposure.  
This system enables open participation while maintaining anonymity.

---

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS

---

## Project Structure
- `index.html` – Application entry point
- `src/` – Main application source code
- `package.json` – Dependencies and scripts
- `vite.config.ts` – Vite configuration
- `tailwind.config.ts` – Styling configuration

---

## Installation & Running Locally

### Requirements
- Node.js (v18+ recommended)
- npm installed

### Steps

1. Clone the repository:
   Click **Code** and Copy the url and paste it in Terminal

2. Navigate to project directory:
   cd repository-name

3. Install dependencies:
   npm install

4. Start development server:
   npm run dev

5. Open the local URL shown in terminal  
   (usually http://localhost:5173)
